算是个总结

发表于 2020-12-31 更新于 2021-01-04 分类于 杂记 阅读次数： Valine：

狗日的 2020 年终于要过去了…… 到了年底的最后一天，写点文字记录一下一年来的得失，也算是个总结吧。

所得

2020 年让我比较满意的是终于完成了自己一直想要写完的两本书：《怎样使用 BRTools》和 《用 LaTeX\LaTeXLATE​X 来轻松写文本》（不是正式出版，自娱自乐而已）。其中前一本书从开始挖坑经历了弃坑、重新填坑、再弃坑、再重填，直到到填坑结束历时有 8 年之久，这本书的完成同时也带来一个副产品，那就是在国家版权局将这个小插件的软件著作权登记成功；后一本书是近两年来开发的 LaTeX\LaTeXLATE​X 宏集 Corp 的使用说明，这个项目居然没有中途弃坑是最令我意外的事情。

2020 年新获得了一些技能，比如说：这个博客的搭建。虽然说这项工作实际上是同事 Roy 手把手帮着完成的，但毕竟是完成了，而且运行正常。另一项技能是终于学会了使用 和 ，从此不再因为不会使用 而错过和一帮大牛一起合作的机会。令人意外的是，单位里有小伙子在院服务器上部署了 Git-Blit，现在我可以在上面肆意地 pull & push 😃 ，我自己多个项目已经放在上面进行 托管（没放到 Github 或 Gitee 上一方面是因为内容不方便公开，另一方面主要是代码太烂）。

正儿八经地使用 Overleaf 也是 2020 年才开始的事，在上面做了一个关于《公路桥梁抗震设计规范》学习的 slides。Overleaf 上绝大多数都是面对科研人员的各种模板，包括各种国际顶级期刊的投稿模板，为了能更好地用一用 Overleaf，我是不是应该好好钻钻业务投几篇 paper 了？

2020 年读的书不多，小林章的《西文字体》、《西文字体 2》，高岗昌生的《西文排版》彻底读了一遍，颇有收获。日本人在这些细节方面的工匠精神还是颇让人赞叹的。什么时候能有中国人出一本关于中文字体的书呢？

非常偶然的情况下，2020 年在 B 站上回顾了一下 1982 年改编自老舍的同名话剧，由谢添执导的电影——《茶馆》，这部电影在初中时曾经看过，那时候太小，感受不深，如今重新观看，被这部不朽作品的力量和老艺术家精湛的表演所震撼了。向老舍先生和于是之、蓝天野、郑榕、英若诚、黄宗洛等一代老艺术家致敬（其实他们的 话剧 更精彩）！

所失

  1. 专业技术水平上还是没有什么提高，兴趣爱好太多，划水摸鱼太多，放在专业的心思上就太少了。
  2. 想要入坑机器学习和深度学习，并且还想在专业方向上有些应用，居然一年之内动都没动，非常不满意……
  3. Perl 语言刚学了个开头，中间就搁置了，做事不能这么没有坚持……
  4. pgfornament-han 一直都没有新的贡献，就靠着 林莲枝 大神独自支撑了，惭愧……
  5. 2019 年写好的一个数据库程序还没怎么应用就被大家弃置了，不过这也说不上是失，Microsoft 的 Access 确实不好用，VBA 是我这辈子再也不想去看的东西了。
  6. COVID-19 推迟了奥运，也推迟了电影《夺冠》的放映，等正式上映时也总是没有合适时间去影院，就错过了这部真正有欲望去看的电影。不过这也不能算是失吧，从后来放出的部分片段来看，也许没有去看是对的，真正催泪的是女排，而不是这部电影。2021，我会继续为女排加油！

所欲

  1. 专业上得用点心思了，想到的几个有意思的问题要好好考虑一下了，关于拱轴线，关于塑性铰……
  2. 周培德的《计算几何——算法设计与分析》至少要啃一两章下来吧，如果做 BIM 的二次开发，一定会有用处的……
  3. python 的科学计算要搞一搞，是坑，但必须要跳。我受够了 Microsoft Excel 了……
  4. 小骆驼书要读完，不能半途而废……
  5. 《营造法式》应该列入 2021 的书单了，从古人的工法中吸取一些经验，顺带检验一下自己的文言文功底还剩几成。
  6. 自己编写的 BRTools 的函数库部分，准备开源扔在 Github 和 Gitee 上了，反正大家都很忙，应该没人关注我的烂代码，万一真有有兴趣的人能 fork 一下还发了 pull request 呢？
  7. 其实还想写一本关于 BRTools 函数库 AutoLisp 代码解析的书，但这是大坑，跳还是不跳……
  8. 另一个是巨坑，LaTeX3 语法。这肯定是 LaTeX 的潮流，而且，随着开发的深入，不会用 LaTeX3 实在是太费劲了。不过，一旦入了坑，以我的重度强迫症来说，应该会把写过的宏集再用 LaTeX3 重构，这是坑中之坑…… 算了，2021 年先不跳了。
感谢拨冗阅读本文，若有些许收获，不妨捐赠以资鼓励。
